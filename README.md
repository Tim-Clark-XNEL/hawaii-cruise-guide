# Hawaii Cruise Guide

This is a lightweight React-based trip page designed to work well on phones and deploy easily to GitHub Pages.

## Files

- `index.html`: entry page
- `app.js`: React app and trip data
- `styles.css`: mobile-first styling

## Quick preview

Because this starter uses browser-based ES modules, you can preview it by opening `index.html` in a browser or serving the folder locally with any static file server.

## Publish to GitHub Pages

1. Create a new public GitHub repository.
2. Upload the contents of this folder.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and the `/(root)` folder.
6. Save, then wait for GitHub to publish the site.

Your site URL will usually look like:

`https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/`
